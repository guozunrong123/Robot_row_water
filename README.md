# RM校内赛——划很多水  电控部分开源

#### 																																									By：郭尊荣

### 关于电硬：

不足之处已在另一文档中指出，总体实现效果较好，没有出现翻车现象。还可以增加一些模块，提高整车性能。有关改进：机械臂电机旋转角度可以考虑采用位置式pid预先设定好，不需要操作手去操作。机械爪考虑采用在两边加红外/激光，遇到块自动夹取后电机旋转到底台高度，提高速度。
